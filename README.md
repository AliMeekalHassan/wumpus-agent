# Wumpus World AI Agent

A simple web-based AI agent that explores a hidden Wumpus World using **logic-based reasoning (Knowledge Base)** instead of random movement.

---

## What it does
- Generates a random grid with pits and a Wumpus
- Gives percepts like **Breeze** and **Stench**
- Uses a Knowledge Base to infer safe cells
- Moves step-by-step based on logic
- Shows real-time grid updates and inference steps

---

## Controls
- **Start** → creates a new world  
- **Step** → moves the agent using logic  

---

## Colors
- 🟩 Safe cell  
- ⬜ Unknown cell  
- 🟥 Dangerous cell
